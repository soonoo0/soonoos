ABOUT US

ST Core Technology는 선도적인 블록체인 기술을 기반으로 금융 플랫폼, 코인 개발, 결제 솔루션 제공 및 블록체인 기반의 게임서비스등을 자체적인 기술력을 통하여 구축해 나가고 있습니다. ST Core Technology의 기술적인 역량으로 안전하고 편리한 금융 거래 환경을 제공하고 있으며, 탁월한 디자인 역량을 통해서 서비스 기초부터 체계적으로 관리되는 Brand Experience 솔루션도 마련되어 있습니다.

ST Core Technology는 기술적인 연관도가 낮은 상이한 기술을 융합하여 창의적인 영역을 확장해 나가면서 블록체인과 ICT개발 영역에서 새로운 가능성을 제시하고 있을뿐만 아니라, 핵심적인 사업에 대한 투명성을 제공하고 블럭체인 시스템을 사용한 결제시스템 인프라를 통하여 제공하고 있는 서비스에서 가져갈수 있는 수익성을 극대화하고 있습니다.

검증가능하고 신뢰할 수 있는, ST Core Technology만의 탄탄한 블록체인 기술을 기반으로, ST Core Technology에서 그간의 경험과 노하우를 통하여 새롭게 재탄생한 iST PAY는 블록체인 기술과 새로운 금융환경이 암호화폐기술로 적응해 가는 사회전반적인 문화영역과 소셜네트워크 영역까지 확장해 나아가는데 있어서 다이아몬드처럼 밝게 빛나며 단단하게 자리잡는 초석이 되어가고 있습니다. 이에 발맞춰서 ST Core Technology는 iST PAY가 블록체인 기술을 통한 거래에서 유래되는 투명성을 보장하고 저비용/고효율로 담보할 수 있는 신뢰성을 확보하면서 건강한 사회를 건설하는데 필요한 사회적 비용을 줄이면서, 부가적인 수익은 다시 사회로 되돌리면서 긍정적인 피드백으로 선순환을 만들어 낼것으로 기대하고 있습니다.

Hyperledger Fabric은 시간당 100건을 상회하는 빠른 트랜잭션 처리가 가능한 블럭체인 시스템이면서 개선가능성이 많았습니다. ST Core Technology에서는 Hyperledger Fabric에 실제 사용자들이 쉽게 접할 수 있도록 편의성을 개선하여 자체적으로 개발한 라이트닝 네트워크를 구축하였고, 현재 특허등록중에 있습니다. ST Core Technology에서 구축하고 제공하는 라이트닝 네트워크에서는 전용 APP을 통하여 안전하면서도 손쉽게 결제와 d송금을 할 수 있으며, 기존의 블럭체인 기반 암호화폐에서는 존재하지 않았던, 실시간에 가까운 금융서비스로서, 다음세대의 금융서비스를 앞당기는 역할을 하면서 서비스를 제공할 수 있게 될것입니다.

Game

ST Core Technology에서는 자체적으로 게임을 기획/개발하고 있으며, 이와는 병렬적으로 다수의 게임 개발사와 협력하여 독자적인 Playstore를 구축하고 있습니다. ST Core Technology자체적인 게임과 Playstore에서는 Ethereum기반으로 새롭게 재탄생한 블록체인기반 시스템과 ERC20, ERC721의 기준을 만족하는 스마트 컨트렉기반으로 구성되어 독자적이고 창의적인 게임컨텐츠를 구성하고 있으며, ST Core Technology에서 마련하고 있는 Hyperledger Fabric기반의 선도적인 지불/결제/송금등의 기능을 사용해서 효율적으로 연동되면서, 게임과 함께 코인을 통한 결제시스템이 자연스럽게 접점을 만들어서 융합되어 나가게 됩니다. 게임과 Playstore를 사용하면서, 사용자가 아이템을 경매에 붙이거나 게임내 캐릭터를 경매에 붙일 수 있고, 이런 연계를 통해서 1:1의 고정적인 게임캐릭터의 가치를 블록체인 기반의 암호화폐가치에 따르는 유동적인 시스템을 바꾸어 낼 수 있습니다. iST 코인 사용자의 경우에는 iST코인의 지분증명방식을 통해 당해 사용자는 랜덤으로 iST 게임의 캐릭터를 받을 수 있습니다. 이것이 의미하는 바는 iST코인의 투자자는 저절로 iST게임 인프라와 연결되면서 게임과 접할 수 있는기회가 주어지게 되고, 반대로 iST게임과 Playstore를 통해 게임을 즐기는 유저는 자연스럽게 iST 코인을 얻고 게임을 통한 자산증식과 부가적인 수익이 생기는 현상이 나타나게됩니다. 결과적으로 블록체인 기반 기술을 통한 게임 시스템이 Hyperledger Fabric iST PAY 시스템과 융합되어, ST Core Technology에서는 문화/게임/SNS/금융 인프라 전반을 아우르는 4차산업의 Melting Pot으로 자리잡을 수 있는 시스템을 선보일 수 있을 것으로 예상합니다.

Mining
Platform

ST Core Technology에서는 진보하고 특별한 Mining Platform을 제공합니다. 제공하는 코인은 자체적으로 운영하는 Private Network 위에서 Proof of Work(PoW) 방식으로 채굴됩니다. ST Core Technology에서는 Mining Platform 서비스를 안정적으로 제공하기 위해서 자체적으로 구축된 Mining Factory를 통해서 코인을 얻고 있으며, 채굴과정 관리감독과 운영은 ST Core Technology에서 담당하고 있습니다. 시스템을 이용하는 사용자입장에서는 간단하게 해시파워와 기간을 정하고 해당하는 비용을 지불하면서 Mining Factory를 Rent하게 되고, ST Core Technology의 관리하게 코인 채굴과정을 모니터링하면서 채굴되는 코인을 가져가실 수 있습니다. Mining Factory를 통해서 만들어진 코인은 사용자가 가져갈수 있으며, 사용자가 편하게 iST 코인이나 ST Core Technology와 MOU 관계의 다른 코인과 교환할 수 있습니다. 결과적으로는 Mining 코인을 사용자가 원하는 코인으로 환전하여, ST Core Technology와 제휴 코인에서 제공하는 코인을 얻을 수 있으며, ST Core Technology 혹은 제휴코인의 ICO에도 간접적으로 참여하여 코인을 취득할 수 있습니다. Mining Factory를 Rent하는 사용자의 입장에서는, 플랫폼에서 생겨나는 부가적인 수익을 지불한 비용만큼 나누어 가져가게 되면서 사용자로서 추가적인 이익을 제공받게 됩니다. ST Core Technology입장에서는 사용자의 증가와 코인의 유통을 통한 선순환을 위해 Mining Factory를 운영하며, 암호화폐와 블럭체인 기반의 생태계가 조성되고 ST Core Technology의 기술력과 선도적인 iST 코인 기반 인프라가 더 많은 역할을 할수 있기를 기원합니다.
